Репозиторій

  • git pull-автоматично отримати зміни та злити віддалену гілку до вашої поточної гілки
  • git add-добавляє усі зміни до стейдж
  • git cimmit-комітить всі зміни
  • git push-аходить на віддалений проект та забирає звідти усі дані, котрих у вас досі нема
  • git checkaut-перейти до нової гілки
